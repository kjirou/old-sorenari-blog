ChromeでnodeアプリをGUIデバッグ

2013年7月18日

Chromeを使ってNode.jsのコードをGUIでデバッグする方法のメモ。
「リモートデバッグ」と呼ばれている。

環境構築と試しに使ってみるまでに少しハマったので、
とりあえず動かすまでの手順をメモだけしときます。

  • node-inspector をインストール
  • $ node-inspector を実行
  • $ node --debug-brk yourfile.js を実行
  • http://127.0.0.1:8080/debug?port=5858 を Chrome から開く
  • そこからの操作は、この記事 や 上記のREADME などを参考にする

ぶっちゃけ、今はシンプルなものしか作ってないので、良さがわかってません。
ただ、前にゲームを作った時に、ひとつの変数を延々追いたいということは良く有ったので、そういう時に使えそう。
