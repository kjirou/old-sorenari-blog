使えるネットが糞重い件について

2007年12月12日

新しく借りた使えるネットVPSが糞重いんだよ！
MySQL5.0のインスコに2.5時間掛かるなんて、喧嘩売ってるんじゃないかと。
freeしてみる

$free -m
total       used       free     shared    buffers     cached
Mem:           256         74        181          0          0          0
-/+ buffers/cache:         74        181
Swap:            0          0          0

な・・・なんでSwapが0なんだ・・・？

ちなみに、前に借りた自分用の環境や他2名の環境を聞いたら、もちろんこんなのじゃなかった。

んー、確かに以前の環境は表示されている性能より良かった気がするけど、これはちょっと困る。
CPUがx86_64とやらに変わってたことも含めて、使えるネットさんで方針の転換でもあったのかしらん。
【追記】
サポートに問い合わせたら、「メモリはSwapを含めて一括管理していて、その中でメモリだけこちらから見れる状態になっていて、Swap領域は見れないけど存在はしている」というような返事を貰った。

なるほど、まぁでも、これで打つ手が見当たらなくなっちまった。
