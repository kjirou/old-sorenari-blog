Amazon SimpleDBがオワコンっぽいので止めたメモ

2015年2月18日

Amazon SimpleDB を使ってみようとしたけど止めた、という経緯のメモです。

1. 管理画面が見つからない

AWS ログイン後に表示される Amazon Web Services のサービス一覧に、該当の項目が無い。

フォーラムを漁ると Where is SimpleDB? という記事が見つかり、どうも無いみたい。

2. zipを展開してサンプルを動かす

管理画面が無いので、とりあえず開発者用のチュートリアルから進めてみた。

途中で、「Amazon SimpleDB Scratchpad」という動作確認用のHTMLのzipを落としてと言われる。

んで、それを Chrome で開いて実行すると Uncaught SecurityError とクライアントJSのエラーが出て動かない。
タブブラウザだとダメらしい？
（iframe間で参照してるから？ Firefoxだと動いたので、未調査放置です）

zipでDLする意味って、なんだろうなぁ・・・。

以上、オワコンっぽい

色々放置されてそうだったので、止めた。
新規で使うなら、おとなしく DynamoDB 使っといた方が良さそう。

自分は、ミニゲーム用の簡単なスコアボードを置ける場所が欲しかっただけなので、parse.com とか別手段を探すことにしました。
