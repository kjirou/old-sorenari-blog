[MySQL] SELECT NOCACHE ではなく SQL_NO_CACHE

2010年2月12日

MySQLで・・・

○正しい構文

SELECT SQL_NO_CACHE ...

×誤った構文

SELECT NOCACHE ...

毎回間違える

毎回間違える上に、
「動かないのは、”NOCACHE”の位置が悪いんじゃないか？」と、
“SELECT NOCACHE” とかでググっても何も出なくてイラっとするので、
検索用にGoogleに拾われることを期待して、記事にしてみました。

ついでに、”SELECT NOCACHE” とか “MySQL NOCACHE” で
ググる人々のイライラ度を下げて、
世界平和に貢献できれば、と考えております。　うそだけど
