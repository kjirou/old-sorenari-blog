git diff が動かない

2012年10月7日

git diff を打っても何も表示されないなー
おかしいなー、怖いな怖いなー と思っていたら
単に $PAGER 環境変数に指定していた lv が無いだけでした
（※サーバは設定したてで別環境からコピった.zshrcを使っていた）

ということで .zshrc を以下の様にして対応

PAGER=less
もしくは
GIT_PAGER=less

存在しないコマンドを指定していても、エラーは出ないんすね

gitを触り始めなモンで
"ワーキングディレクトリ"や"インデックス"とか
この辺の理解が誤ってて出力されないのかと若干ハマったよこん畜生
