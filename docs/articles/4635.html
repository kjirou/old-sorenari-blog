npm selenium-webdriver-util

2015年8月12日

https://github.com/kjirou/selenium-webdriver-util

という、selenium-webdriver 用のユーティリティセットを作りました。

selenium-webdriver 本体の要素セレクタがあまりにも貧弱なので、主にその辺を補助するものになります。

俺のように、権威と実績を重視して E2E テストフレームワークに selenium-webdriver を選択したけど、割りと不便で思ったより不安定なんだけど、でも今更構成を変えたくないけどどうにかならないかな、という後ろ向きに前向きな方々に少しでも幸せを届けられたらと思います。

Description に “loosely coupled utilities” と書いている通り、thenable を受けて promise を返すだけのような、疎結合な静的な関数の集まりなので、後付でも大丈夫だと思います。
