web-environmentsというnpmパッケージを作った

2013年7月19日

任意の設定値や環境変数を定義するためのライブラリです。
例えば、Base URL や API Key などの、
環境により変わる値をまとめて定義するためのものです。

正直、var envs = {}; の中に入れるのと大差ないですが、
ちょっとひねって、こういうアクセスを可能にしてみたりしています。

var baz = webenv.get("foo.bar.baz");

ダウンロードや使い方などは GitHub リポジトリ を参照いただければ幸いです。
