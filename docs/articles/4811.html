Elona無印派に贈るoomExの解説

2016年12月22日

Elona を無印以外でプレイしたことが無い方に向けて、omake 系列バリアントの究極系である oomEx（= omake_overhaul_modify ex）を紹介する内容です。

（Roguelike Advent Calendar 2016 の 22 日目の記事です。）

Elona とは

もしかすると Elona 自体をご存じない方が読むかもしれないので、一応書いときます。

Elona とは、noa猫 氏が開発した、フリーのローグライクRPGです。
対応 OS は Windows 、言語は HSP です。

実はそれほどローグライクではなくて、ターン制の行動・自動生成ダンジョン・プレイに適切な状況判断が求められる、というローグライク要素はありつつも、全体的なゲーム感としては「自由度が高く、何でも出来る中でハック＆スラッシュ要素が高い MMO」といった趣です。
個人的には、Ultima Online（例えが古い）のプレイ感覚に似てるなぁって思ってます。

なお、詳細が気になった場合は、YouTube や ニコ動 などの動画サイトを検索してプレイ動画を観ると良いと思います。

バリアント とは

そんな Elona ですが、開発は数年前に終了しています。

安定版である 1.16final のリリースが 2009/05/17、開発版と呼ばれる非安定版である 1.22 のリリースが 2010/05/08 のようです。いずれにせよ、大昔です。

そんな状況で、細かい経緯はわかりませんが、noa猫氏本人の意志でソースコードが開示され、オープンソースのように有志の方による開発が今でも進んでいます。

バリアントとは、その有志の方それぞれが独自に拡張した Elona それぞれのことです。

oomEx とは

Elona omake バリアントの拡張である、
Elona omake overhaul バリアントの拡張である、
Elona omake overhaul modify バリアントの拡張である、
Elona omake overhaul modify ex バリアントのことです。

有志の方の拡張を別の有志の方が独自に引き継ぐ、まるでロマサガ２の継承法のような体です。
この中で oomEx だけは、今でもゆるやかにですが開発が継続されています。

特徴として、この omake から派生したバリアント全般に言えることなのですが、

・操作のストレスを減らす（多少簡単になっても）
・煩雑な作業を減らす（多少簡単になっても）
・Elona 本体のゲーム感は、なるべく損なわないようにする

このような趣旨で開発されていることです。oomEx はその究極形です。

機能紹介: ダメージポップアップ

omake から oomEx に至るまで、数多くの改善が行われてきましたが、自分がその中で特に嬉しかったのはこのダメージポップアップ機能です。

どういう機能かというと、YouTubeから拝借してきた動画の 14:12 頃から御覧ください。

「ハクスラなのに、ダメージ値や特殊効果が出なくて強くなった感覚が味わいにくい」

・・・という、Elona 最大の矛盾を解決してくれました！

例えばこれで、序盤で調達できる、黄金の騎士＆★大地の大槌 が如何に頼りになるかが、すごくわかり易くなりました。

今までは、何か強そうな武器を取っても、どれだけ強いのかわからなかったんですよね・・・。
（毎回サンドバックを叩きに行けば別ですが）

バリアントのススメ

とても紹介しきれないので諦めてますが、それはもうたくさんの改修がなされています。

本家こそ至高という考えは個人的にはとても良くわかるのですが、一度でいいのでバリアントを試してみると、きっと楽しいと思います。

oomEx まではちょっと・・・という方は、overhaul 以外にも色々なバリアントの元になっていて、準公式的な扱いを受けている omake をとりあえず入れてみるのがオススメです。
