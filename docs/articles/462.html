[PHP] microtimeがfloat型に対応してた

2010年3月18日

PHP4時代

microtime は、文字列しか返さなかったので

list($micro, $sec) = explode(" ", microtime());
$sec = intval($sec);
$micro = floatval($micro);

わざわざexplodeで分割して、
各々どうにかするような処理を書いていた記憶があります。

PHP5時代

どうやら、第一引数に $get_as_float という引数が付いていたようで・・・

$micro = microtime(true);
var_dump($micro); // -> float(1268908179.96)

floatで取得できるようになってたのを、教えてもらった。
