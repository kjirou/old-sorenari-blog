[JavaScript] IEのsplit(Regex)は空値を消す

2012年2月12日

// Firefox
>>> ','.split(/,/).length //-> 2
// IE
>>> ','.split(/,/).length //-> 0 ..空文字列は削除、 /,/g 的動き
// 全ブラウザ
>>> ','.split(',').length //-> 2

つまり split(正規表現) ではなくて split(文字列) にしとけというイマサラなメモ

参考) 正規表現で文字列を split すると、できた配列から空の要素が消える
