MarkdownとreST記法でのリンク書式比較

2013年2月26日

GitHub Wikiなどで使われているマークダウン記法と
Sphinx などで使われている reStructuredText 記法の、それぞれでのリンクの書き方

比較っつうか、毎度忘れて仕方ないのでメモ

Markdownでのリンクの書き方

[リンクテキスト](URL)
[それなりブログ](http://blog.kjirou.net/)

補足:

  • 別ウィンドウ(target="_blank")で開く方法は無い

reStructuredTextでのリンクの書き方

`リンクテキスト <URL>`_
`それなりブログ <http://blog.kjirou.net>`_

補足:

  • これも別窓で開く方法は無い
  • 上記は外部リンクに対しての書き方、内部リンクに関しては別途専用のロールを使う
