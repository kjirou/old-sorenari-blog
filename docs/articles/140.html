ぶちまけた

2007年9月25日

使えるネットにApache2.2でも入れようかと思って

cd /usr/local/src
wget http://ftp.kddilabs.jp/infosystems/apache/httpd/httpd-2.2.6.tar.gz
tar fvxz httpd-2.2.6.tar.gz
cd httpd-2.2.6
./configure --prefix=/usr/local \
--enable-so
make
make install

あああああ、志村！ prefix！！ ぷれふぃっくすー！！！
