HTTPリクエスト確認用WebサーバをPythonで作った

2013年3月1日

たまーにですが、HTTPリクエストを生に近い形で確認したいことがあるので、
投げられたHTTPリクエストをそのまま表示するだけの、簡単なWebサーバを作りました。

インストールと使い方

インストール:

$ pip install mahokanta

サーバ起動:

$ mahokanta

この状態で、ブラウザで localhost:8080 にアクセスすると、
ブラウザとコンソールの両方に、HTTPリクエストの内容が表示されます。
Python 2 / Python 3 / PyPy に対応！

パッケージやソースなど:

  • PyPI/mahokanta
  • GitHub/python-mahokanta
