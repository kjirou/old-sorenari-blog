今年、やって良かったと思ったこと

2008年12月31日

酒が入って全然まとまる気がしないので、項目を箇条書きだけで。

1. 社内用とか公開用とか色々なCentOSサーバを建てまくったこと。
サーバ構築に関しては、とりあえず小規模Webサービス用ならどうにか出来る気がする。

2. Webサービスを最初から最後まで面倒見て作ったこと。
ものって完成するんだなーと思った。

3. デスマを捌いたこと
収束させることが出来て、割とマシな野郎だなぁと思った（自分を）。
一生の内一回位は経験してもいいかもだけど、
IT業界に居れば勝手に向こうから降ってくるので、進んでやろうとは思わんが。

4. Python & Django
実際に公開まで行ったので、この構成なら「普通のWebサービスなら大丈夫」という感じになった。
一回受託案件で使ってみることが出来ると、もう少しいいかも。
後、Pythonについては体系的な勉強をほとんどしてないので、本の通し読みとかをしとかんと。

あー、他にも書きたかったけど、
年が明けそうなんでこの辺で。

それでは皆様、良いお年を。
