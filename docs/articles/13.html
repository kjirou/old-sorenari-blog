mysqlコマンドにパスワードを含めることができるのか

2007年4月17日

mysqlコマンドで
今までは↓の様に記述していて

mysql -u username -p password

パスワードを認識してくれないので
そういうものかと諦めてました。

したら
職場の小金を溜めて海外旅行に行くのが趣味の奴に

mysql -uusername -ppassword

こう書けばいけますよ

と、言われてやってみたら、認識してくれた。

MySQLを知って2年の間
疑問に思ったことがなかったっす。
