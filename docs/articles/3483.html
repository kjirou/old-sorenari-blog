Macでdotfileを表示/非表示にするシェルコマンド

2013年7月22日

Mac で、Finder やデスクトップ内の
ドットファイル・ドットディレクトリ（？）の表示と非表示を切り替えるコマンドです。

これを .bashrc や .zshrc にコピペして:

showallfiles ()
{
  if [ "$1" = "" -o "$1" = "on" ]; then
    defaults write com.apple.finder AppleShowAllFiles TRUE
  else
    defaults write com.apple.finder AppleShowAllFiles FALSE
  fi 
  killall Finder
}

bash や zsh からこんな風に使う:

$ showallfiles on  # 表示
$ showallfiles off  # 非表示
