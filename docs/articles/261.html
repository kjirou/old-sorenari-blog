[Flex] Alertのボタンやカレンダーを日本語化

2008年2月6日

Flexの小ネタもいいところ！
書かないよりはマシなんじゃね？ってことで。
【Alert.showのボタンのラベルを日本語に変える】
デフォルトで 「Yes」「No」なので、「はい」「いいえ」に変えたいときは

Alert.yesLabel = "はい";
Alert.noLabel = "いいえ";

こうやって、クラス変数を直接書き換えて変える！

【DateChooserやDateFieldの曜日や月を日本語に変える】
例えば、DateFieldの曜日を変えるとき

var df:DateField = new DateField;
df.dayNames = ["日", "月", "火", "水", "木", "金", "土"];

月を変えるときは、monthNamesを変更する。
