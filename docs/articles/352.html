プログラム用のWeb日本語リファレンスまとめ

2008年9月11日

自分が良く使っている
Webから閲覧できるプログラム言語用リファレンスのまとめ。
特に理由は無いが、簡単に書けそうな記事なので書く。

【PHP】
PHP マニュアル
本家。PHPを使ってて、ここを知らない人は居ないでしょうけど。
サイトはたまに重いけど概ね良好、検索があって使い易い。
言語自体もマニュアルぐらい頑張ればいいのに、って、つい褒めちゃう程。

【Ruby】
Ruby リファレンスマニュアル
本家。全文検索が無くなったのは痛かった。（でも、そのためしょっちゅう落ちてたけど）
マニュアルのDLは可能なので、もっと便利に使いたかったら自分でカスタマすればいい。

タイトル不明
本家のマニュアルをベースに使い易くまとめてくれている。

逆引きRuby
まんま、やりたいことから逆引きしたい場合はいい。
Rubyを一番最初に触ったときは、ここを一度上から流して目を通して置いて、
後で「これあったな」という感じで参照してた。

【Python】
Python ドキュメント
本家・・・かな？2.4迄対応している。
基本的には「グローバルモジュールインデクス」からトップダウン形式で検索している。
ちなみに組み込み関数は __builtin__ ってリンクを押した先にある。

Django オンラインドキュメント和訳
ほとんどここで事が足りる。更新もちょっぱや。

【JavaScript】
JavaScriptリファレンス
一世代前の怪しげ＆適当なJavaScript解説サイトでは無く、
JavaScriptを言語として解説してくれている数少ないリファレンス。

jQuery 開発者向けメモ
jQuery日本語リファレンス
今、jQueryを使いまくっている中でお世話になっているサイト。
下は美味しいところだけをわかり易く解説してくれている、1.2.6対応。
どちらかといえば上の方が充実はしているが、対応は1.2.1迄。

【ActionScript3】
Flex 3 リファレンスガイド
Flex 2 リファレンスガイド
本家。システマチックにまとめている割には重いし使い難い。
（まるでAS3そのものだ・・・と感じたのは内緒）
でも、これしか無いので仕方なし。

【Bash】
Bashリファレンス
シェルスクリプト用。情報量が多くまとまりも良い。

【HTML/CSS】
とほほのWWW入門
説明不要！

英語嫁っていうのもあるけど、だったらソース読んだ方が手っ取り早い。
日本語は日本語で価値があると思う。
