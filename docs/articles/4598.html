画像を分割するimage-divider

2015年4月22日

image-divider という、設定ファイルを参照して一枚の画像から複数の画像を切り取る npm モジュールを作りました。
RPGツクールやウディタのアイコンセットから、アイコンを分割するときなどに使います。

クライアントソフトを使わなかったのは、コマンドラインから扱いたいからです。
シェルスクリプトじゃなくて npm モジュールにしたのは、gulp の間入れて本番では Data URI Schema に変換したりなど、柔軟に扱えるようにするためです。

それでも、要件を満たすものが絶対あると思って1時間位探してたんですが、見つからなかったので作りました。
類似のもっといいやつをご存じの方、是非教えてプリーズ願います！
