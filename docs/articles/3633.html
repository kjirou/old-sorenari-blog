[Pixelmator] 範囲を選択してコピー＆ペースト

2013年8月13日

Mac アプリ Pixelmator の画像内でのコピペ操作についてです。
バージョンは 2.2

自分の場合、開いた画像の形式にもよるのかもですが、
単純に Cmd + C ＞ Cmd + V で出来なかったのでメモしときます。
超普通の操作のはずなのに、すぐわからなかった・・・。

  • まず、メニューバー「表示」＞「レイヤーを表示」で、レイヤーを見れるようにしておく
  • おそらくは左側にある「ツール」の 矩形選択 からコピーしたい場所を選択して、Cmd + C して Cmd + V
  • レイヤーがひとつ増えているので、それを選択
  • 「ツール」の 移動 で移動する
  • メニューバー「レイヤー」＞「すべてのレイヤーを統合」で、まとまる

直接同じレイヤーにコピペする方法は不明でした。
絶対あるはずだと思って結構調べたのですがみつからず..
