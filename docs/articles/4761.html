リバーシゲーム作成用ライブラリ npm reversi

2015年12月20日

reversi という npm ライブラリを作りました。

https://github.com/kjirou/reversi

いわゆる、オセロゲーム・リバーシゲームのコアロジックをまとめたライブラリです。
AIは含まれておりません。本人も弱いのに作れるはずがない。

一応、プレイテスト用のCUIコマンドもつけています。
操作はかなり辛いので、気合を入れて下さい。

ちなみに、今働いているところで、「30分で作るオセロゲーム大会」というのがあり、
その時は全然ダメだったんですが、そのときの残パワーで作りました。

今更、使い道があるのかは激しく疑問ですが、
誰か使っていただけたら激しく喜びます。
