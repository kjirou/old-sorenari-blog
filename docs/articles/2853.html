[jQuery] Cannot set property ‘cur’ of undefined

2013年6月9日

jQuery で、animate やそのサブメソッドの fadeIn, fadeOut などを使った際に

Cannot set property ‘cur’ of undefined

というエラーが出たら、とりあえず最新版の jQuery-1.10 系に更新するといいかもしれません。

詳細は不明ですが、自分は jQuery-1.9.1 でこれが出た時に、
jQuery-1.10.1 へアップデートしたら直りました。
