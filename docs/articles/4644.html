CircleCI REST API ラッパーの npm torus

2015年8月26日

こちらです：

https://github.com/kjirou/torus

このように使います：

$torus -t your_api_token project kjirou your-repo-name
success 2015-08-25T09:44:09.747Z https://circleci.com/gh/kjirou/your-repo-name/87 master 42
success 2015-08-25T09:12:46.239Z https://circleci.com/gh/kjirou/your-repo-name/86 master 40
success 2015-08-25T05:02:48.911Z https://circleci.com/gh/kjirou/your-repo-name/85 master 41
success 2015-08-24T19:28:23.703Z https://circleci.com/gh/kjirou/your-repo-name/84 master 62
...

作った目的：

「朝出社すると、夜走らせているCIが何回か落ちている」
・・・という状況がままあり、その Failed だったビルド一覧を手軽に抽出するためです。

とりあえずは、今現在欲しいこの機能だけが存在します。
今後もやりたいことが増えたら追加しますが、CircleCI REST API がイマイチであるという理由から、もはや無い可能性もあります。

また、急ぎで作ったために作りが非常に雑です。
CI 用のツールなのにテストねぇとか、もうこれわかんねぇな・・・。
