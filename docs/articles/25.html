ココを読んで、!!(変数) は簡潔で良いなぁと思ったので、利用に向けてまとめてみる
PHP(4)でも出来るのかなーと思ったら出来たので、こっちからまとめる。 
<?php
class Blank {}
class NotBlank {var $a = 1;}
$b = "";
$vals = array(
0,               // false
1,               // true
0.0,            // false
0.1,            // true
"",             // false
"0",            // false
"1",            // true
"0.0",          // true
".",            // true
"a",            // true
array(),        // false
array(0),       // true
(new Blank),    // false
(new NotBlank), // true
"$b",          // false
"{$b}",        // false
null,           // false
true,           // true ・・・一応
false,          // false ・・・一応だって
);
foreach ($vals as $v) {
var_dump(!!$v);
}
