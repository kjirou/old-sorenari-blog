rpgmaterial.js JRPG風ゲームコンテンツ作成用ライブラリ

2012年10月12日

rpgmaterial.js

JRPG風なゲーム、またはWebコンテンツを作成するための
JavaScriptライブラリを公開しました

　　rpgmaterial – Github

とりあえずは、Webで出来るRPGツクールを目指していて
現在含まれている主な機能は以下です

  • マップエディタ
  • 会話ダイアログ
  • キャラチップ操作
  • HTTPリクエスト操作
  • 内部計算用クラスや共通関数など

全く未完成な状況で
そもそもエンジニア以外の人に向けて作り始めたのですが
現在は、内部APIしかありません
これから、ゲーム作りついでにボチボチ足して行きます

よろしければ見てやって下さいまし！
