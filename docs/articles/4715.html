npm mocha-automatic-coffeemaker

2015年10月4日

mocha-automatic-coffeemaker を npm publish しました。

テスト対象ファイルと同じパスの構造で、./test 以下にテストコードの雛形を生成するコマンドです。

こんな風に使う：

$ tree ./lib 
lib
└── utils.js
$ npm install -g mocha-automatic-coffeemaker
$ genmochatest lib/utils.js
Generated successfully!
root: /home/username/your-project
src : lib/utils.js
test: test/lib/utils.js
$ cat test/lib/utils.js
describe('lib/utils', function() {
});
