[PHP] PHPでSoftbankの絵文字コードをソースに書く

2008年2月29日

その道の人には常識なんだろうけど、割とハマった。

ソフトバンク用の絵文字は
普段使わないような文字を含む5バイトのコードで
テキストエディタとかから入力は普通できないような文字を使っているため
絵文字一覧ページ等からコピーして張り付けるのが一般的な配置方法らしい。

ただ、PHPソースにコピペすると、
PHPが解釈できない文字が含まれているようでエラーになってしまった。

なので、16進表記の記法を使って書く

// 「家」の絵文字
$emoji_ie = "\x1B\x24GV\x0F";
// 「電話」の絵文字
$emoji_denwa = "\x1B\x24G)\x0F";

“\x1B” と “\x24” と “\x0F” は固定なので、
後は絵文字一覧のページから残り2文字の部分を確認して、書き換えて文字を変えられる。
